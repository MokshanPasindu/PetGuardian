# PetGuardian Backend

Spring Boot REST API for the PetGuardian AI-powered pet health and safety system.

## Features

- 🔐 JWT Authentication & Authorization
- 🐾 Pet Management (CRUD operations)
- 📋 Medical Records & Vaccination Tracking
- 🤖 AI Skin Analysis Integration
- 🏥 Veterinary Clinic Discovery
- 📅 Appointment Booking
- 💬 AI Chatbot
- 👥 Community Posts & Comments
- 📱 QR Code Pet Identification

## Tech Stack

- Java 17
- Spring Boot 3.2
- Spring Security + JWT
- Spring Data JPA
- MySQL / H2 Database
- Swagger/OpenAPI Documentation

## Prerequisites

- Java 17+
- Maven 3.8+
- MySQL 8.0+ (for production)
